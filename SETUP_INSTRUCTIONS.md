# Flick Messenger - Инструкции по установке и запуску

## 📦 Что входит в проект

- **client/** - React компоненты и страницы
- **server/** - Express сервер (для production)
- **shared/** - Общие типы и константы
- **package.json** - Зависимости проекта
- **pnpm-lock.yaml** - Lock файл для зависимостей
- **vite.config.ts** - Конфигурация Vite
- **tsconfig.json** - Конфигурация TypeScript

## 🚀 Быстрый старт

### 1. Распаковать архив
```bash
tar -xzf flick-messenger-src.tar.gz -C /путь/к/папке
cd /путь/к/папке
```

### 2. Установить зависимости
```bash
pnpm install
# или npm install
# или yarn install
```

### 3. Запустить dev сервер
```bash
pnpm dev
# Приложение откроется на http://localhost:5173
```

### 4. Собрать для production
```bash
pnpm build
pnpm start
```

## 🎯 Основные функции

✅ **Чаты** - список всех чатов и контактов
✅ **Профиль** - редактирование профиля пользователя
✅ **Настройки** - все настройки приложения
✅ **Архив** - архивирование чатов через свайп влево
✅ **Контакты** - управление контактами
✅ **Меню** - плавная анимация меню
✅ **Темы** - светлая и темная тема

## 📱 Как использовать свайп функцию

### На телефоне:
1. Нажмите и удерживайте палец на чате
2. Сдвиньте палец влево (примерно на 50-100 пикселей)
3. Нажмите красную кнопку "Архив"
4. Чат переместится в архив

### На ПК (мышкой):
1. Нажмите левую кнопку мыши на чате
2. Удерживая кнопку, сдвиньте мышку влево
3. Отпустите кнопку мыши
4. Нажмите красную кнопку "Архив"
5. Чат переместится в архив

### Вернуть из архива:
1. Перейдите в вкладку "Архив"
2. Свайпните чат влево (как выше)
3. Нажмите зеленую кнопку "Вернуть"
4. Чат вернется в основной список

## 🎨 Дизайн

- **Стиль**: Telegram-style (минималистичный, чистый)

- **Шрифты**: System fonts (Inter fallback)

## 📝 Примечания

- Все данные сохраняются в localStorage
- Приложение работает полностью на клиенте (static frontend)
- Нет необходимости в backend для базовой функциональности
- Для production используйте `pnpm build` и `pnpm start`

## ❓ Часто задаваемые вопросы

**Q: Как добавить новый чат?**
A: Используйте вкладку "Контакты", введите юзернейм и нажмите "Добавить"

**Q: Как изменить профиль?**
A: Нажмите на аватар в левой панели, отредактируйте поля и нажмите "Сохранить изменения"

**Q: Как архивировать чат?**
A: Свайпните чат влево и нажмите красную кнопку "Архив"

**Q: Как вернуть чат из архива?**
A: Перейдите в вкладку "Архив", свайпните чат влево и нажмите зеленую кнопку "Вернуть"
